<h1 align="center">Aplikasi clone KAI Access</h1>

## Resources
-   React Native
-   Expo
-   Supabase
-   react-navigation

### How to run in localhost

1. clone project
```
git clone https://github.com/Handika743/Project-clone-app-KAI.git
```
2. Install dependencies
```
npm install
```
3. start demo app
```
expo start
```
