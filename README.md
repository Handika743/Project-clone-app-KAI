<h1>Website Sistem Data Pegawai</h1>

<h2>Resources</h2>
-   React Native
-   Expo
-   Supabase
-   react-navigation

<h2>How to run in localhost</h2>

<h3>1. clone project</h3>
```
git clone https://github.com/ryanabi-RA/Project-clone-app-KAI.git
```
<h3>2. Install dependencies</h3>
```
npm install
```
<h3>3. start demo app</h3>
```
expo start
```
